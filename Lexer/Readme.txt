Our Lexer is created using a lex file. The Makefile that is included will do 
everything for you, so using the make command will tokenize our example 
program and output those tokens and values to both the standard output and 
PS_Lexer_Output.txt.
We used an Ubuntu command line to run and compile the files, so the version of lex
that we used is the one that comes from the command: sudo apt-get install flex
